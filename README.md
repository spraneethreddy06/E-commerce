# 🛒 E-Commerce Website

A responsive front-end E-Commerce website built using **HTML, CSS, and JavaScript**. The project simulates the core functionality of an online shopping platform, including user authentication, product browsing, shopping cart management, checkout, order tracking, and an admin dashboard.

## 🚀 Live Demo

## 📌 Features

- 🏠 Responsive Home Page
- 👤 User Registration & Login
- 📦 Product Listing
- 🔍 Product Details Page
- 🛒 Shopping Cart
- 💳 Checkout
- 📜 Order History
- 🛠️ Admin Dashboard
- 📱 Responsive Design
- ⚡ Interactive User Interface
